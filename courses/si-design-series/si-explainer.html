01
Foundation

Signals Are Electromagnetic Fields,
Not Electrons

Before anything else in signal integrity, this concept must be internalised: a signal is not electrons moving through a copper trace. A signal is an electromagnetic field propagating through the dielectric between two conductors. The copper sets the boundary conditions — it does not carry the energy.

In a PCB transmission line, two conductors work together: the signal conductor and the return reference conductor. Both are equally necessary. Remove either one and the electromagnetic field is no longer guided — it spreads into space and becomes a source of EMI.

A signal propagates as an EM field in the dielectric between signal trace and return plane. Both conductors are required to contain the field. Forward current flows in the trace; return current flows simultaneously in the plane beneath it.

The signal wavefront — the leading edge of the electromagnetic transition — is where all the high-frequency energy is concentrated. The faster this transition (the shorter the rise time), the more energy is compressed into a shorter physical distance. This is why rise time, not clock frequency, is the governing parameter in signal integrity.

Signal bandwidth from rise time
BW = 0.35 / tr
tr in ns · BW in GHz
1 ns rise time → 350 MHz bandwidth
100 ps rise time → 3.5 GHz bandwidth
Why this matters for EMC

Every SI problem and every EMI problem in a digital PCB originates from the same root cause: electromagnetic fields that are not properly contained between their two conductors. Solve the containment problem and you solve both simultaneously.

02
Return Path

The Return Path
Is Part of the Signal

Return current is not an afterthought. It is half of the transmission line. The electromagnetic field that constitutes the signal is formed between the forward current path and the return current path — if the return path is poorly designed, the signal is poorly designed.

Return current does not simply flow to the nearest ground connection. Its path depends on frequency. At low frequency, the inductive component of impedance is small, so return current spreads across the entire return plane, following the path of least resistance. At high frequency, inductance dominates — and the path of least inductance is the one directly beneath the signal trace, minimising the area of the current loop.

At low frequency, return current spreads across the plane. At high frequency, it concentrates directly beneath the signal trace, minimising loop area and inductance. Anything that interrupts this path — a slot, gap, or split — forces the return current to detour, increasing loop area and EMI.

What Breaks the Return Path

Gaps, slots, and splits in the return reference plane force high-frequency return current to detour around the obstruction. The current loop area increases dramatically — and radiated emissions scale with loop area. This is one of the most common causes of EMC test failures.

Routing a signal trace over a plane gap is functionally equivalent to routing it with no return plane at all at high frequency. The fields are uncontained, the impedance discontinuity creates reflections, and the enlarged current loop radiates.

✓ Correct

Solid return reference plane beneath every signal layer. No splits, no cuts, no large voids. Return vias placed immediately adjacent to signal vias at every layer transition.

✗ Avoid

Split planes beneath high-speed signals. Routing over plane gaps. Layer transitions without nearby return vias. "Isolated ground islands" connected by narrow bridges.

03
Impedance

Controlling Characteristic Impedance

Characteristic impedance is not a material property — it is a geometric property of the transmission line. It depends on the width of the signal trace, the distance to the return reference plane, and the dielectric constant of the material between them.

When a signal transitions from one impedance to another — at a via, a connector, a trace width change, or a reference plane gap — part of the signal reflects back toward the source. The reflected energy creates ringing, overshoot, and timing errors at the receiver.

Microstrip (outer layer, one reference plane) vs Stripline (inner layer, two reference planes). Stripline offers better field containment and lower crosstalk but cannot be inspected visually after assembly.

Vias as Impedance Discontinuities

Every via on a high-speed signal trace creates a capacitive stub — the cylindrical barrel of the via acts as a shunt capacitor. At frequencies where the stub length approaches a quarter wavelength, the stub resonates and can completely attenuate the signal. This is why back-drilling (removing the unused via barrel) is standard practice on multi-gigabit boards.

A companion return via must be placed immediately adjacent to every signal via at a layer transition. Without it, the return current must travel along the plane to the nearest ground via before it can transition — the detour creates inductance and a partial impedance break.

04
Termination

Reflections, Ringing,
and When to Terminate

A signal launched into a transmission line travels at a finite speed. When it reaches the load, if the load impedance does not match the line impedance, part of the signal reflects back toward the source. This reflected wave and the original signal superimpose, creating the characteristic ringing waveform seen on oscilloscopes.

Not every trace needs termination. A line is electrically short when its propagation delay is less than approximately one-sixth of the signal rise time. Below this threshold, reflections return to the source before the signal transition is complete, and the effect is negligible.

When a line becomes electrically long
lcritical = tr / 6 × vp
vp ≈ 6 in/ns in FR4 · tr in ns
1 ns rise → terminate lines longer than 1 inch
500 ps rise → terminate lines longer than 0.5 inch
Series termination places a resistor at the source — quieter launch, no DC power. Parallel termination places a resistor at the load — clean signal everywhere on the line but draws continuous current. Choice depends on topology, fanout, and power budget.
Termination myth

Adding termination "just in case" to short traces does not improve signal quality — it degrades it. A correctly terminated short line wastes power and can slow edges. Only terminate when the trace exceeds the critical length for the given rise time.

05
Crosstalk

How Signals
Interfere With Each Other

Crosstalk is the unintentional transfer of energy from one transmission line (the aggressor) to an adjacent one (the victim). It occurs because the electromagnetic field of the aggressor is not perfectly contained between its two conductors — some field couples into the victim line through both capacitive (electric field) and inductive (magnetic field) mechanisms.

The two types produce different effects. Capacitive coupling injects current into the victim in both directions. Inductive coupling injects voltage. At the near end (same end as the aggressor driver), these effects partially add. At the far end, they partially cancel — but never completely, and their relative magnitudes depend on the coupling geometry and termination.

NEXT (Near-End CrossTalk) appears at the same end as the aggressor driver. FEXT (Far-End CrossTalk) appears at the far end. The 3W rule — maintaining a trace separation of at least 3 times the trace width — is the primary layout tool for crosstalk reduction.

Practical Rules for Crosstalk Control

R1Apply the 3W rule. Keep edge-to-edge trace separation ≥ 3× trace width. This contains approximately 70% of the electric field between conductors, leaving less to couple into adjacent traces.
R2Route on adjacent layers orthogonally. Signals routed in the same direction on adjacent layers have maximum parallel coupling length — the worst case. Orthogonal routing on adjacent layers reduces crosstalk to near zero.
R3Use buried microstrip or stripline for sensitive signals. The second reference plane on the opposite side of a buried trace reduces the field that couples laterally, decreasing both NEXT and FEXT.
✗Do not route high-speed and low-speed signals in parallel over long distances. Even at 3W spacing, a sufficiently long parallel run will accumulate significant coupled energy.
✗Do not use split planes to isolate analogue and digital returns. The split creates a gap that enlarges return current loops for signals that cross it — EMI increases even as the perceived isolation is meaningless at high frequency.
06
Noise

Ground Bounce
and Plane Noise

When multiple output drivers switch simultaneously — a common event in buses, memory interfaces, and FPGAs — each driver draws a fast transient current through the power delivery path. The inductance of the IC package and the PDN (power delivery network) converts this di/dt into a voltage spike.

Voltage spike from switching inductance
ΔV = Lpkg × di/dt
Lpkg = package + via inductance (typically 1–10 nH)
10 nH × 1 A/ns → 10 V spike on supply rail
This spike corrupts adjacent signals and can cause logic errors

This spike appears on both VCC and GND rails simultaneously — hence the name "ground bounce." From the perspective of the output driver, the ground reference shifts momentarily, causing the driven signal level to appear to change when it has not. Adjacent inputs may misinterpret this as a valid transition.

The solution is to reduce the inductance seen by the switching current. This means placing decoupling capacitors as close as possible to the IC power pins, using low-inductance power delivery paths (closely spaced plane pairs), and choosing ICs with lower package inductance where critical.

Connection to SI

Ground bounce is where power integrity and signal integrity meet. A clean signal launched by a driver can be corrupted before it leaves the package if the power delivery is noisy. Solving PDN is not separate from solving SI — they are the same problem viewed from different angles.

07
Routing

Routing Rules for
Signal Integrity

Routing for signal integrity is routing for controlled fields. Every decision — layer choice, spacing, via placement, corner geometry — is a decision about where the electromagnetic field will be and whether it will be properly contained.

01Never route over a gap in the return plane. The return current must detour, the loop area increases, and the impedance is no longer controlled. This single rule prevents more EMC failures than any other.
02Place a return via within 2× the dielectric thickness of every signal via. At a layer transition, the return current needs a local path to follow the signal to the new layer. Without a nearby return via, it detours along the plane — adding inductance.
03Match the reference plane on both sides of a layer transition. If a signal moves from a GND-referenced layer to a PWR-referenced layer, the return current must cross from GND to PWR — place a decoupling capacitor close by to provide that path.
04Route differential pairs tightly and with matched length. Tightly coupled pairs share the same field region, improving common-mode rejection. Skew (length mismatch) converts differential signal energy to common-mode, which cannot be rejected.
05Minimise serpentine meander amplitude. When length-tuning with serpentines, keep the spacing between meander segments at least 3× the trace width to avoid coupling the forward segment into the return segment.
✗90° corners are a myth — but stubs are not. A 90° corner in a trace at digital speeds does not cause measurable signal degradation. What does cause problems is an unterminated stub — a branch off a signal trace that creates a parallel resonator.

Board Partitioning

High-speed digital, analogue, power conversion, and RF sections of a board each generate and are sensitive to different types of interference. Placing them in dedicated zones — with careful routing discipline at the boundary — prevents common-impedance coupling. The return reference plane beneath each zone should be continuous; isolation comes from zone placement, not from splitting the plane.

08
Summary

Eight Rules to Apply
Before Your Next Layout

01Think in fields, not wires. Signal integrity is a field containment problem. Every rule that follows is a consequence of this.
02Rise time sets the rules, not clock frequency. Calculate BW = 0.35/tr before choosing trace geometry, stackup, and termination strategy.
03Protect the return path above all else. Solid return planes. No gaps beneath high-speed signals. Return via at every layer transition.
04Match impedance throughout the signal path. From driver output impedance, through the trace, to the load — any discontinuity reflects energy.
05Terminate only electrically long lines. A trace longer than tr/6 × vp needs termination. Shorter traces do not — termination on short lines degrades signal quality.
06Apply the 3W rule to parallel traces. Minimum edge-to-edge spacing of 3× trace width reduces lateral field coupling to an acceptable level.
07Route adjacent layers orthogonally. Eliminates parallel coupling between layers entirely — the most powerful and lowest-cost crosstalk fix available.
08Solve PDN and SI together. Ground bounce corrupts signals before they leave the IC. A strong PDN is a prerequisite for signal integrity, not a separate discipline.
