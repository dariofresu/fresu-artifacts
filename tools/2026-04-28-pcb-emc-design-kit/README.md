# PCB EMC Design Kit

**Date:** 2026-04-28  
**Type:** Interactive HTML Tool  
**Status:** Published  

## Summary

Standalone HTML tool built from the PCB Design Mastery lecture series.  
Three-tab interface: Cheatsheet, Checklist, and Self-Assessment.

## Contents

- `pcb-emc-design-kit.html` — Standalone interactive tool (open in any browser)

## What's inside

**Cheatsheet**
- 10 Golden Rules (Critical / Warning / Info, colour-coded)
- Return current behaviour by frequency table (DC → low → high)
- 8 key terminology cards (RRP-first language throughout)

**Checklist**
- Tier 1 — Critical (9 items): RRP integrity, return vias, plane continuity
- Tier 2 — Warning (9 items): routing, placement, separation
- Live score bar + per-section counters + reset

**Self-Assessment**
- 8 questions drawn from PCB Design Mastery lecture series
- Immediate feedback per question (correct explanation + wrong explanation)
- Dot progress tracker + scored result with tier label
