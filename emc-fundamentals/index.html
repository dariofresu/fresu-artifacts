Lumped vs. Distributed Elements
Why rise time — not clock frequency — governs PCB signal behaviour
Core Principle

A trace transitions from a lumped element to a distributed element when its propagation time exceeds one quarter of the signal rise time. This single threshold is the root cause of reflections, ringing, crosstalk, and EMI — independent of clock frequency.

Fig 1 — Left: slow rise time compared to propagation time keeps all energy in a lumped bath — no reflections. Right: fast rise time creates a travelling wave; reflections appear at every impedance discontinuity.
L_crit = 0.25 × t_rise × v_prop   |   v_prop = c / √εr
Common Misconception

Clock frequency does not determine EMI risk — edge rate does. A product running at 500 kHz with fast-edge ICs can fail EMI testing. A product at 100 MHz with controlled edge rates may be perfectly clean. All EMI, all crosstalk, and all reflections occur during the rising and falling edge — nothing happens between transitions.

Rule 01

Find Rise Time Before Designing

Rise and fall time data is not in IC datasheets. It is in IBIS and SPICE models — ASCII files you can open. Look for dV/dt (slew rate) and derive t_rise. You cannot design correctly without knowing this value.

Rule 02

Match Propagation Time, Not Length

When matching bus lines, match propagation time — not physical length. Outer layer traces propagate ~15% faster than inner layers. Outer traces must be longer to match timing with inner layer routes.

Rule 03

Audit Die Shrink Replacements

Pin-compatible replacement ICs often have faster edge rates due to process shrinks. A board that worked flawlessly can fail EMI testing after re-qualification with a faster part at the same clock frequency.

Rule 04

Design for Termination Readiness

Even without buying controlled-impedance boards, route traces so that termination resistors can be added at the correct locations without a respin. This is insurance against future component changes.

Energy & Fields
Where electromagnetic energy actually travels in a PCB — and why it matters for EMC
The Fundamental Reality

All electromagnetic energy in a PCB transmission event travels in the dielectric material between conductors — not through the copper. The copper acts as a waveguide: the lowest-impedance boundary that steers the fields from source to load. Voltage is the integral of the E-field over distance. Current is electron movement driven by that field.

Fig 2 — Cross-section of a microstrip. E-field lines (green) propagate entirely within the FR4 dielectric. Field density is maximum under the trace centreline and falls off laterally. Return current flows in the surface of the reference plane, induced by the field.
Principle 01

Energy Lives in the Fields

There is no energy in voltage or current — both are measurable effects of the electromagnetic field. The field in the dielectric carries all energy. Once understood, every EMC design decision follows logically from this single fact.

Principle 02

Copper Steers — It Does Not Carry

The trace and reference plane form a waveguide. Energy attaches to the copper boundary because it presents the lowest impedance. Remove the return plane and the fields spread outward until they find one — causing radiation and EMI.

Principle 03

Path of Least Impedance — Always

Energy always takes the path of lowest impedance — 100% of the time, no exceptions. Understanding where that path is at your operating frequency is the entire discipline of EMC engineering.

Principle 04

You Route Half a Transmission Line

Every trace you route is one conductor of a two-conductor transmission line. The return path is the other conductor. Neglect it and the fields will find their own return path — almost never the one you intended.

Return Current Behaviour
How return current distribution changes with frequency — and why this defines your EMC performance
At DC and very low frequencies, return current distributes broadly across the reference plane following the path of lowest resistance — the shortest direct path.
Two-Layer Board Implication

On two-layer boards without a ground plane, low-frequency traces use each other as return paths. The only way to channel this return current is to route a dedicated return trace — at least 3× the signal trace width — directly adjacent from source to load. This constrains the field spread and reduces coupling between circuits.

Above audio frequencies, return current concentrates in the reference plane directly under the signal trace. This is the path of minimum inductance (smallest magnetic field volume) and maximum capacitance (highest E-field density) simultaneously.
Why Directly Below?

The path directly under the trace simultaneously minimises loop inductance (smallest magnetic field volume) and maximises capacitance (densest E-field). These two effects make it the lowest-impedance path at any frequency above a few hundred hertz on typical PCB dimensions. Any plane split, void, or anti-pad forces this current to detour — dramatically increasing loop area and radiation.

Ground Plane Splits — Critical Warning

Splitting a reference plane under a high-frequency signal forces return current around the gap, creating a large current loop. A large current loop is an efficient radiating antenna. Any IC application note recommending split ground planes around mixed-signal devices should be treated as incorrect until verified against first-principles measurements. The correct solution is a solid plane with the analog and digital circuits carefully placed — not a split.

Reference Plane Design
How your reference plane configuration directly determines radiated emission performance

The same signal trace routed above four different reference plane configurations will produce four different EMI signatures. The configuration determines how tightly return current is channelled and how well fields are contained.

Best Solid Reference Plane

Uninterrupted copper plane provides continuous low-impedance return. Fields tightly contained. Return current flows directly under the trace. Always the target configuration.

Moderate Crosshatch Plane

Partial copper coverage provides an imperfect return path. Fields spread through voids. Used in flex circuits. Acceptable only where a solid plane is not feasible.

Poor Split Reference Plane

The gap forces return current around the split, creating a large loop area. That loop radiates. Never route high-frequency signals across a plane split.

Worst No Reference Plane

Without a reference plane, fields expand in all directions searching for a return conductor. Maximum loop area, maximum inductance, maximum EMI. Acceptable only for very low-frequency, low-speed designs.

Inductance, Proximity & Stack-up Rules
The Proximity Rule

Proximity — not conductor size — is the primary determinant of inductance. Moving conductors closer together reduces the magnetic field volume, which reduces its inertia. To halve inductance by widening traces alone, you would need to increase trace width by 50×. Moving the conductors 2× closer achieves the same or greater reduction with no width change.

Stack-up

Signal Layers Adjacent to Planes

Always route critical distributed-length signals on layers directly adjacent to a reference plane. Thinner dielectric = tighter coupling = lower inductance = better field containment.

Decoupling Vias

Side-by-Side Via Placement

Place decoupling cap vias side-by-side (not at opposite ends). Closer via spacing means smaller current loop = lower inductance. Route the vias as close to the IC supply pin as geometry allows.

Layer Selection

Inner vs Outer for Critical Signals

Route distributed-length high-frequency signals on inner layers (stripline) for better field containment. Use outer layers only when the trace is short enough to be a lumped element — avoiding the via overhead.

App Notes

Verify Before Implementing

IC application note layout recommendations should be assumed incorrect until verified against first principles. App note authors understand circuit theory — not always PCB layout physics. Check every recommendation against return current and field behaviour.
