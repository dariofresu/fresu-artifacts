Lumped vs. Distributed Elements
Why rise time — not clock frequency — determines your PCB behaviour
Core Principle

A PCB trace transitions from a lumped element to a distributed element when its propagation time exceeds one quarter of the signal's rise time. This transition is the root cause of reflections, ringing, crosstalk, and EMI — regardless of clock frequency.

Fig 1. — When propagation time exceeds ¼ of rise time, energy becomes a travelling wave. Reflections, ringing and EMI follow.
Lcrit = 0.25 × trise × vprop  |  vprop = c / √εr
Common Misconception

Engineers often monitor clock frequency to assess signal integrity risk. This is incorrect. A board clocking at 500 kHz with fast-edge ICs can fail EMI testing while a board at 100 MHz with carefully controlled edge rates may be perfectly clean. Edge rate is everything.

Rule 01

Find Rise Time First

Rise and fall time data is not in datasheets — it is in IBIS and SPICE models. Open the ASCII model file and look for slew rate (dV/dt), then calculate trise.

Rule 02

Match Time, Not Length

When matching bus lines, match propagation time. Outer layers propagate ~15% faster than inner layers — outer traces must be proportionally longer to match timing.

Rule 03

Beware Die Shrinks

When an IC undergoes a die shrink, edge rates increase. A board that worked perfectly can fail after a component re-qualification if the replacement part has faster edges.

Rule 04

Design for Termination

Even if you don't buy controlled-impedance boards, design traces so termination resistors can be added at the right locations without a board respin.

Energy, Fields & Transmission Lines
Where energy actually travels in a PCB — and why it matters for EMC
The Fundamental Truth

All electromagnetic energy travels in the dielectric medium between conductors — not through the copper. The copper acts as a waveguide, steering the fields because it presents the lowest impedance path. Voltage and current are measurable effects of the fields, not the source of energy.

Fig 2. — Cross-section of a microstrip. The electromagnetic energy (E-field shown as ↕) travels entirely in the FR4 dielectric between trace and plane. Copper conducts surface currents but does not carry the energy.
Principle 01

Energy is in the Fields

All energy in a PCB transmission event is carried by the electromagnetic field in the dielectric. Voltage is the integral of the E-field; current is electron movement driven by the field.

Principle 02

Copper is a Waveguide

Trace + plane form a waveguide. Energy attaches to copper because it is the lowest-impedance boundary. Remove the return plane and the fields spread until they find one — often with catastrophic results.

Principle 03

Path of Least Impedance

Energy always takes the path of lowest impedance — always, 100% of the time, no exceptions. Understanding where that path is at your operating frequencies is the foundation of EMC engineering.

Principle 04

You Route Half a Line

Every trace you route is only half of a transmission line. The return path is the other half. Ignoring it means the fields will find a return path for you — usually not the one you want.

Return Current Behaviour
How return current distribution changes with frequency — and why it defines your EMC performance
At DC and very low frequencies, return current distributes broadly across the reference plane — taking the path of lowest resistance.
Low Frequency Design Rule

On 2-layer boards without a ground plane, traces at low frequencies use each other as return paths. The only way to channel low-frequency return current is to route a dedicated return trace 3× the width of the signal trace, directly adjacent from end to end.

Above audio frequencies (~1 kHz on typical boards), return current concentrates directly under the trace — minimising loop area and inductance.
Why directly below?

The path directly under the trace simultaneously minimises the magnetic field volume (lowest inductance) and maximises the electric field density (highest capacitance). This is the lowest impedance path — so that is where the current flows. Any plane split, void, or discontinuity forces this current to divert, spreading the fields and causing radiation.

Critical Warning — Ground Splits

Splitting a reference plane under a high-frequency signal trace forces return current around the split, massively increasing the loop area. This creates a highly efficient antenna. Any IC application note recommending split ground planes around mixed-signal devices should be treated with extreme scepticism and verified against first principles.

Reference Plane Design
How your reference plane configuration directly determines radiated emission performance

A trace routing above four different reference plane configurations will produce four different EMI signatures. The configuration determines how well return current is channelled and how tightly fields are contained.

BEST Rank #1

Solid Reference Plane

Uninterrupted copper plane provides continuous, low-impedance return path. Return current flows directly under the trace. Fields are maximally contained. Always the target configuration.

MODERATE Rank #2

Crosshatch Ground Plane

Used in flex circuits and some RF designs for controlled impedance with flexibility. Provides a partial return path. Fields partially spread through the voids. Not ideal for high-frequency digital signals.

POOR Rank #3

Split Reference Plane

Any gap or split in the reference plane forces return current to route around it, creating a large current loop. This loop acts as an antenna. EMI radiation increases dramatically. Avoid crossing signals over plane splits.

WORST Rank #4

No Reference Plane

Without a reference plane, fields spread in all directions searching for a return path. The result is maximum loop area, maximum inductance, and maximum EMI radiation. Acceptable only for very low frequency, low-speed circuits.

Inductance & Proximity Rules

The Proximity Rule

Proximity, not conductor size, is the primary determinant of inductance. Moving conductors closer together reduces the magnetic field volume, which reduces its inertia (inductance). To reduce inductance by 50% by increasing trace width alone, you would need to go from 10 mil to 500 mil width. Moving them closer achieves the same or better result with no width change.

Stack-up Rule

Signal/Plane Adjacent Pairs

Always route critical signals on layers that are directly adjacent to a reference plane. The tighter the coupling (thinner dielectric), the lower the inductance and the better the field containment.

Via Rule

Decoupling Cap Via Placement

Place vias side-by-side (not at opposite ends of the capacitor pad). Closer via spacing = smaller loop area = lower inductance. Vias should be as close to the IC supply pin as geometrically possible.

Layer Rule

Inner vs Outer for Critical Signals

For distributed-length signals at high frequency, route on inner layers (stripline) for better field containment. Use outer layers (microstrip) only for lumped-length elements or where the via overhead exceeds the benefit.

Power Plane Rule

Referencing Power vs Ground

A trace may reference a power plane under specific conditions. The AC impedance of a power plane at high frequency is essentially the same as a ground plane — provided there is a low-impedance connection between them at the point of interest via decoupling capacitors.
